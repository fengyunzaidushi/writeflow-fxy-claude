## 📝 文档规范

### 文档命名
- 使用小写字母和连字符
- 例如：`slidev-ppt-guide.md`

### 文档格式
- 使用 Markdown 格式
- 包含目录（TOC）
- 提供代码示例
- 添加相关链接

### 更新维护
- 标注最后更新时间
- 记录版本信息
- 保持内容同步
